## Dynamic Programming의 조건
+ 작은 문제가 반복이 일어나는 경우
+ 같은 문제는 구할 때마다 정답이 같다.

---------
## 구현방법
### 1. Bottom-up
+ 아래서부터 시작해 구하는 방법
+ 작은 문제부터 차근차근 구해나가는 방법

### 2. Top-down
+ 위에서부터 문제를 풀어나가는 방법
